# 🧠 Neuro Abraço

> Projeto acadêmico desenvolvido no 1º semestre de Análise e Desenvolvimento de Sistemas da **FMU – São Paulo**, na disciplina de **Front-End com UX/UI**.

---

## 💙 Sobre o Projeto

O **Neuro Abraço** é um site informativo voltado para pessoas autistas e suas famílias, com foco em promover **bem-estar, inclusão e qualidade de vida**.

O projeto reúne informações sobre:

- 🧩 **Níveis de suporte no TEA** (DSM-5)
- 📍 **Locais de atendimento especializado** em São Paulo
- 🌳 **Atividades inclusivas e de lazer** na cidade
- 🥦 **Orientações alimentares** para pessoas autistas
